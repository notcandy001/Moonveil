## Required Packages

<details>
<summary>üì¶ Dependencies</summary>

Moonveil relies on a small, intentional set of tools.  
Install the following packages for the setup to work as intended.

## üì¶ Pacman Packages
### Core
- **Hyprland** ‚Äì Wayland compositor
- **Waybar** ‚Äì Status bar
- **Rofi** ‚Äì Application launcher
- **Hyprlock** ‚Äì Lock screen
- **Wlogout** ‚Äì Logout / power menu
- **SwayNC** ‚Äì Notification center
- **gnome-bluetooth-3.0**
- **vte3**
- **imagemagick**

### Shell & CLI
- **Zsh** ‚Äì Default shell
- **power-profiles-daemon**

### Utilities
- **Grim** ‚Äì Screenshot utility
- **Nautilus** ‚Äì File manager
- **Pavucontrol** ‚Äì Audio control

### Theming & Appearance
- **LXAppearance** ‚Äì GTK theme manager

### Python Dependencies
- **python**
- **python-gobject**
- **python-psutil**
- **python-watchdog**
- **python-pillow**
- **python-toml**
- **python-ijson**
- **python-numpy**
- **python-requests**

### Fonts
- **JetBrainsMono Nerd Font**
- **Noto Fonts Emoji**
- **Noto Fonts CJK**

## üì¶ Yay (AUR) Packages

### Core
- **Matugen** ‚Äì Dynamic color generation (**required**)
- **python-fabric-git**
- **fabric-cli**
- **python-setproctitle**
- **gray**

### Shell & CLI
- **Oh My Zsh** ‚Äì Zsh framework
- **Powerlevel10k** ‚Äì Zsh prompt theme
- **Eza** ‚Äì Modern `ls` replacement

### Theming & Appearance
- **adw-gtk-theme** ‚Äì GTK theme
- **Bibata Modern Ice** ‚Äì Cursor theme

### Fonts
- **Geist Mono (OTF)** ‚Äì Primary UI & terminal font
- **Geist Mono Nerd Font** ‚Äì Icon support
- **PP Neue Machina** ‚Äì Display / clock font
